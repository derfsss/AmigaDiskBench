# AmigaDiskBench v2.2.13 - Visualization Enhancements Plan (Approved)

## Overview
Enhance the "Visualization" tab with multi-series support, hybrid charts (MB/s + IOPS), 16 high-contrast colors, and a dynamic wrapping legend.

## Feature Roadmap

### 1. New Chart Types
- **Scaling Profile (P1)**: Line chart of Block Size vs MB/s.
- **Drive Battle (P2)**: Grouped Bar chart comparing Volumes.
- **Stability Trend (P3)**: Line chart of Time vs MB/s.
- **Workload Mix (P4)**: Bar chart of Test Type vs IOPS.
- **Hybrid Profile (P5)**: Overlay of MB/s (Bars) and IOPS (Line) for a single drive.

### 2. Architecture Updates
- **16-Color Palette**: Optimized for high-contrast on Amiga displays.
- **Legend Wrapping**: Automatic multi-line support for complex datasets.
- **Series Grouping**: "Color By" option for Drive, Test, or Block Size.

## Technical Goals
- Modularize `gui_viz_render.c` into chart-specific drawers.
- Implement robust data aggregation in `gui_viz.c`.
- Ensure low-overhead rendering using standard `graphics.library` primitives.
